MongoDB学习笔记

1. Linux环境配置MongoDB

1.1 下载和安装

在MongoDB官网下载tgz文件，注意Version的格式x.y.z中y应选择偶数（稳定版）。上传文件至服务器。

解压文件tar -zxvf mongodb-linux-x86_64-ubuntu1804-5.0.2.tgz

移动文件夹mv mongodb-linux-x86_64-ubuntu1804-5.0.2 /usr/local/mongodb

新建几个目录，分别用来存储数据和日志、配置

mkdir -p /usr/local/mongodb/data
mkdir -p /usr/local/mongodb/log
mkdir -p /usr/local/mongodb/conf

新建并修改配置文件

vim /usr/local/mongodb/conf/mongod.conf

内容如下：

systemLog:
  #MongoDB发送所有日志输出的目标指定为文件
  destination: file
  path: "/usr/local/mongodb/log/mongod.log"
  logAppend: true
storage:
  #mongod实例存储其数据的目录
  dbPath: "/usr/local/mongodb/data"
  journal:
    #启用或禁用持久性日志以确保数据文件保持有效和可恢复。 
    enabled: true
processManagement: 
   #启用在后台运行mongos或mongod进程的守护进程模式。 
   fork: true
net:
   #服务实例绑定的IP，默认是localhost 
   bindIp: 0.0.0.0
   port: 27017

1.2 启动MongoDB服务

cd /usr/local/mongodb/
bin/mongod -f conf/mongod.conf
# 启动失败试试这条
bin/mongod -f conf/mongod.conf --repair

如果启动后不是 successfully ，则是启动失败了。原因基本上就是配置文件有问题

使用navicat连接MongoDB：

若连接不上则关闭服务器的防火墙

# 安装防火墙(bushi)
# apt install firewalld firewall-config
# 查看防火墙状态
systemctl status firewalld
# 临时关闭防火墙
systemctl stop firewalld
# 开机禁止启动防火墙
systemctl disable firewalld

1.3 关闭MongoDB服务

mongo --port 27017 
# 切换到admin库 
use admin 
# 关闭服务 
db.shutdownServer()

##### 或者直接kill
ps -ef|grep mongo
kill -s 9 pid

1.4 数据修复

一旦数据损坏，则需要进行如下操作

删除lock文件

rm -f  /usr/local/mongodb/data/*.lock

修复数据

/usr/local/mongdb/bin/mongod --repair --dbpath=/usr/local/mongodb/data

2. 数据库的创建和删除

命令行进入mongodb,远程服务器键入mongo

查看当前数据库信息：

show dbs ->admin 0.000GB config 0.000GB local 0.000GB（默认三个）

新建数据库:

use articledb -> 直接使用use创建新数据库

查看正在使用的数据库:

db

MongoDB 删除数据库：

db.dropDatabase()

提示：主要用来删除已经持久化的数据库

3. 集合的创建和删除

集合，类似关系型数据库中的表。 可以显示的创建，也可以隐式的创建

显式创建

db.createCollection("myCollection")

隐式创建

当向一个集合中插入一个文档的时候，如果集合不存在，则会自动创建集合,通常我们使用隐式创建文档即可

查看集合
show collections 
或
show tables
删除集合
db.集合名称.drop()

如果成功删除选定集合，则 drop() 方法返回 true，否则返回 false。

4. 文档基本CRUD

文档（document）的数据结构和 JSON 基本一样。 所有存储在集合中的数据都是 BSON 格式。和mysql一样可以单条插入也可以多条插入。

单条插入

使用insert() 或 save() 方法向集合中插入文档

db.collection.insert( 
    <document or array of documents>, 
    { 
        writeConcern: <document>, 
        ordered: <boolean> 
    }
)

示例【隐式创建】：

db.comment.insert({
    "articleid": "100000",
    "content": "今天天气真好，阳光明 媚",
    "userid": "1001",
    "nickname": "Rose",
    "createdatetime": new Date(),
    "likenum": NumberInt(10),
    "state": null
})

查询刚才插入的数据：

db.comment.find()
