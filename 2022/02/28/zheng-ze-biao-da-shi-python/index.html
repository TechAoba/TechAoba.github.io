正则表达式python

发表于2022-02-28|更新于2022-03-01|python
|阅读量:

初见

以下为python的普通字符串匹配：

pattern1 = "cat"
pattern2 = "bird"
string = "dog runs to cat"
print(pattern1 in string)       # True
print(pattern2 in string)       # False

同样的例子，我们使用regex库实现匹配功能：

import re                # regex包

pattern1 = "cat"
pattern2 = "bird"
string = "dog runs to cat"
print(re.search(pattern1, string))       # <re.Match object; span=(12, 15), match='cat'>
print(re.search(pattern2, string))       # None

字符正则化

  • 中括号选择形式

    正则匹配串的形式为 r"regexStr"，即字符串前加r。

    正则化完成单字符的多模式匹配。r"r[au]n"，可匹配ran或者run：

    # multiple patterns("run" or "ran")
ptn = r"r[au]n"
print(re.search(ptn, "dog runs to cat"))    # <re.Match object; span=(4, 7), match='run'>

  • 中括号范围形式

    当希望匹配更多情况时，上面的方式未免太麻烦了。其实中括号匹配还提供了字符范围的表示方法，r"r[A-Z]n"，这样就可以把括号内的字符替换为A到Z的任意字符啦！[0-9a-z]既可以匹配数字也可以匹配字母

    print(re.search(r"r[A-Z]n", "dog runs to cat"))  
# None
print(re.search(r"r[a-z]n", "dog runs to cat, and cat ran"))
# <re.Match object; span=(4, 7), match='run'>
print(re.search(r"r[0-9]n", "dog r2ns to cat"))      
# <re.Match object; span=(4, 7), match='r2n'>
print(re.search(r"r[0-9a-z]n", "dog runs to cat"))  
# <re.Match object; span=(4, 7), match='run'>

  • \表示转译，因此要匹配反斜杠需要使用\\:

    # \\ : match \
print(re.search(r"runs\\", "runs\ to me"))         
# <re.Match object; span=(0, 5), match='runs\\'>

  • .可以匹配除\n以外的任意字符串：

    # . : match anything (except \n)
print(re.search(r"r.n", "r[ns to me"))        
# <re.Match object; span=(0, 3), match='r[n'>

预定义字符集

  • 数字，通配符：\d

    # \d : decimal digit
print(re.search(r"r\dn", "run r4n"))        
# <re.Match object; span=(4, 7), match='r4n'>

  • 非数字字符，通配符：\D

    # \D : any non-decimal character
print(re.search(r"r\Dn", "run r4n"))        
# <re.Match object; span=(0, 3), match='run'>

  • 匹配空白键，\s可以匹配tab、空行等:[\t \n \r \f \v]

    # \s : any white space [\t \n \r \f \v] 
print(re.search(r"r\sn", "r\nn r4n"))           
# <re.Match object; span=(0, 3), match='r\nn'>

  • 匹配分空白键的字符，\S:

    # \S : opposite to \s, any non-white character
print(re.search(r"r\Sn", "r\nn r4n"))           
# <re.Match object; span=(4, 7), match='r4n'>

  • 所有字母、数字和’_’，\w:

    # \w : [a-zA-Z0-9]
print(re.search(r"r\wn", "r\nn r4n"))      
# <re.Match object; span=(4, 7), match='r4n'>

  • \W与\w相反：

    # \W : opposite to \w
print(re.search(r"r\Wn", "r\nn r4n"))      
# <re.Match object; span=(0, 3), match='r\nn'>

边界

  • \b表示空白字符，但只能放在头和尾。\b代表单词之间看不见的转移字符，如here is a word，这句中有好几个\b，每个单词前后都有一个\b。因此\b适合用在匹配整个单词的时候，如果只是单词中的一部分则匹配不了：
print(re.search(r"\bruns\b", "dogs runs to cat"))       # <re.Match object; span=(5, 9), match='runs'>
print(re.search(r"\br\buns", "dogs r unsto cat"))       # None
print(re.search(r"\bruns", "dogs runsto cat"))          # <re.Match object; span=(5, 9), match='runs'>

  • \B就是反过来，代表非单词间：

    print(re.search(r"\Bruns\B", "dog runs to cat"))          # None
print(re.search(r"\Bo\B", "dog runs to cat"))             # <re.Match object; span=(1, 2), match='o'>

  • ^匹配字符串头:

    # ^ : match line beginning
print(re.search(r"^dog", "dog runs to cat"))     
# <re.Match object; span=(0, 3), match='dog'>

  • ^和flags=re.MULTILINE完成多行头匹配

    # multi-line
string = """
dog runs to cat.
I run to dog.
"""
print(re.search(r"^I", string))                         # None
print(re.search(r"^I", string, flags=re.MULTILINE))     # <re.Match object; span=(18, 19), match='I'>

  • $匹配字符串尾：

    # $ : match line ending
print(re.search(r"cat$", "dog runs to cat"))  
# <re.Match object; span=(12, 15), match='cat'>

数量词

  • *：0次或者无限次

    # * : occur 0 or more times
print(re.search(r"ab*", "a"))           
# <re.Match object; span=(0, 1), match='a'>
print(re.search(r"ab*", "abbbb"))       
# <re.Match object; span=(0, 5), match='abbbb'>

  • +：1次或者无限次

  • {m}:匹配前一个字符m次，也可以匹配字符串

  • {n,m} ： 出现n到m次。注意里面不能有空格!

    # {n,m} : occure n to m times
print(re.search(r"ab{2,10}", "ab"))         
# None
print(re.search(r"ab{2,10}", "abbbbbbb"))   
# <re.Match object; span=(0, 8), match='abbbbbbb'>
print(re.search(r"a(bc){2,5}d", "abcbcbcd"))
# <re.Match object; span=(0, 8), match='abcbcbcd'>
# 匹配字符串bc

  • ?0次或者1次，(str)?，比如面对一些简称时可能有字符串的一段也可能没有：

    # ? : may or may not occur
print(re.search(r"Mon(day)?", "Monday"))        
# <re.Match object; span=(0, 6), match='Monday'>
print(re.search(r"Mon(day)?", "Mon"))       
# <re.Match object; span=(0, 3), match='Mon'>

逻辑、分组

  • group组可以把匹配字符串的部分信息进行分组：

    # group
match = re.search(r"(\d+), Date: (.+)", "ID: 021523, Date: Feb/12/2021")
print(match.group())                    
# 021523, Date: Feb/12/2021
print(match.group(1))                   
# 021523
print(match.group(2))                   
# Feb/12/2021

  • 当组数量非常大时，使用?P<groupName>对组进行命名是非常有必要的：

    # use ?P<groupName> named group
match = re.search(r"(?P<id>\d+), Date: (?P<date>.+)", "ID: 021523, Date: Feb/12/2021")
print(match.group())                    
# 021523, Date: Feb/12/2021
print(match.group('id'))                  
# 021523
print(match.group('date'))                   
# Feb/12/2021

  • | ： 表示逻辑或

    print(re.findall(r"(ran|run)", "run ren ran"))    
# ['run', 'ran']
print(re.findall(r"r(a|u)n", "run ren ran"))      
# ['u', 'a']

一些函数

  • 使用findall寻找所有匹配

    print(re.findall(r"r[au]n", "run ren ran"))     
# ['run', 'ran']

  • 替换re.sub() replace

    print(re.sub(r"r[au]ns", "catches", "dog runs to cat"))     
# dog catches to cat

  • 分裂：

    # re.split()
print(re.split(r"[,;\.]", "a;b,c.d;e"))         
# ['a', 'b', 'c', 'd', 'e']

  • 编译：

    # compile
compile_re = re.compile(r"r[au]n")
print(compile_re.search("dog ran to cat"))
# <re.Match object; span=(4, 7), match='ran'>
